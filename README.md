Chris' Eclipse Utility plugins
=====================

This repository is a dumping ground for handy utility plugins I have created. Including

* **Eclipse test fragment creator** - A context menu action on plugin projects to create a test fragment project for the plugin. A wizard guides the user through steps to create the new project.
